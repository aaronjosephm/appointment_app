About SipherCatta:

Our number one priority is to ensure that your privacy is secure. We use the latest encryption technology to prevent any data leakage/hacking.

iOS platform

SipherCatta is now available for iOS. It is a fast ready to use app!

1. The master flow

In the app the user has access to any messaging platform they wish to use! This gives the user complete control over which communication medium they wish to send encrypted messages over.

2. Settings

The user has access to both RSA, and OTP encryption. In the settings tab the user can choose which list to view, and from there can edit the list accordingly.

3. Dealing with keys

The user can generate new keys, and delete them as well. The user can also add new keys by scanning a QR code, and can also send keys via email, airdrop, etc.

4. Encryption

There are two types of encryption - RSA, and OTP. The user chooses which type he/she wants to use before encrypting a message. The user then selects the key, enters the message in the box, and presses the 'Encrypt' button. An encrypted message will appear and also be copied to the paste board!

5. Decryption

Decryption is easy. Once the user pastes the encrypted message into the box, the key is detected automatically. If the user doesn't have the correct key for decrypting the message then they are notified.
